# leafy-xmenu

⚠️ **Status: In Development**  
APIs, behavior, and features are unstable and may change without notice.

leafy-xmenu is a minimal, lightweight X11 menu written in C using XCB.  
Inspired by dmenu, focused on simplicity, speed, and low dependencies.

## Features
- Pure XCB (no Xlib)
- Keyboard-driven menu
- Override-redirect window (WM independent)
- Fast startup, low memory
